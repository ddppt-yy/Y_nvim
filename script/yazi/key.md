Yazi 的快捷键深受 Vim 启发，以高效的单键操作为主。我把它们分门别类整理了出来，方便你随时查阅。

### 🌳 导航 (Navigation)

| 快捷键             | 功能说明                      | 备注     |
| :----------------- | :---------------------------- | :------- |
| `h` / `Left`       | 返回上一级目录                |          |
| `l` / `Right`      | 进入选中的目录                |          |
| `j` / `Down`       | 向下移动光标                  |          |
| `k` / `Up`         | 向上移动光标                  |          |
| `gg`               | 移动到列表最顶部              |          |
| `G`                | 移动到列表最底部              |          |
| `H`                | 返回上一个访问的目录 (后退)   |          |
| `L`                | 前进到下一个访问的目录 (前进) |          |
| `z`                | 使用 `zoxide` 快速跳转目录    |          |
| `Z`                | 使用 `fzf` 模糊查找目录或文件 |          |
| `Ctrl` + `o` / `i` | 后退/前进 (类似 Vim 习惯)     | 可自定义 |
| `Tab`              | 显示当前文件的详细信息        |          |

### 📂 文件操作 (File Operations)

| 快捷键    | 功能说明                             | 备注 |
| :-------- | :----------------------------------- | :--- |
| `y`       | 复制 (Yank) 选中的文件或目录         |      |
| `x`       | 剪切 (Cut) 选中的文件或目录          |      |
| `p`       | 粘贴 (Paste) 已复制的文件或目录      |      |
| `P`       | 粘贴 (覆盖)，若目标文件已存在则覆盖  |      |
| `d`       | 删除 (移至回收站/Trash)              |      |
| `D`       | **永久删除**，**不会**进入回收站     |      |
| `r`       | 重命名选中的文件或目录               |      |
| `a`       | 创建新文件或目录 (以`/`结尾表示目录) |      |
| `o`       | 打开选中的文件                       |      |
| `O`       | 以交互方式打开选中文件               |      |
| `;` / `:` | 运行一个 Shell 命令                  |      |

| `v`          | 进入可视模式 (Visual Mode)，用于连续选择 |      |
| `V`          | 可视模式 (非设置模式)，行为可能不同      |      |
| `Ctrl` + `a` | 选中**所有**文件和目录                   |      |
| `Ctrl` + `r` | 反向选择所有文件和目录                   |      |
| `Esc`        | 取消所有选择                             |      |

### 📋 复制路径 (Copy Paths)

| `cf`   | 复制当前选中文件的文件名 (不含路径) |      |
| `cn`   | 复制文件名 (不含扩展名)             |      |

### 🔍 搜索 (Search)

### ⚡ 进阶操作

除了基础快捷键，Yazi 还提供了一些强大功能：

* **批量重命名**：选取多个文件后按 `r`，Yazi 会用你系统设定的编辑器 (`$EDITOR`) 打开一个文本文件，在里面编辑好新旧文件名列表，保存退出即可一次性批量重命名。
* **压缩与解压**：需要通过插件或自定义快捷键实现。例如，可以在 `keymap.toml` 中将 `C` 键绑定到 `ouch` 插件来压缩文件。
* **与系统剪贴板交互**：默认 `y`、`x`、`p` 操作的是 Yazi 内部的缓冲区。要使用系统剪贴板，通常需要安装 `system-clipboard.yazi` 这类插件。
* **自定义快捷键**：所有快捷键都可以在 `~/.config/yazi/keymap.toml` 文件中自定义。例如，你可以在其中添加 `g d` 来快速跳转到下载文件夹。

需要提醒的是，表格里依赖 `fd`、`fzf`、`ripgrep`、`zoxide` 等外部工具的功能，需要提前在系统里安装好才能使用。

大部分快捷键直接按键即可生效，部分需要组合键（如 `Ctrl`+`a`）或连续按键（如 `gg`、`g d`），留意一下停顿就好。
